STEPS TO PERFORM:
================

2. INSTALL DEPENDENCIES
   npm install

3. BUILD THE PROJECT
   npm run build

4. VERIFY BUILD OUTPUT

   - Verify dist/images/zeal-logo.jpg exists
   - Ensure dist/index.html exists

5. DEPLOY TO NETLIFY (RECOMMENDED)
   Option A - Using Netlify CLI:
   - Install netlify-cli: npm install -g netlify-cli
   - Deploy: netlify deploy --prod --dir=dist
   
   Option B - Manual upload:
   - Go to https://app.netlify.com/drop
   - Drag and drop the dist/ folder
   - Get the assigned netlify.app domain

6. CONFIGURE CUSTOM DOMAIN
   In Netlify dashboard:
   - Go to Site settings → Domain management
   - Add custom domain: shop.zedseclabs.co.za
   - Note the DNS records provided by Netlify

7. UPDATE GODADDY DNS
   Required DNS records for shop.zedseclabs.co.za:
   
   Option A (CNAME - Recommended for Netlify):
   Type: CNAME
   Name: shop
   Value: [your-netlify-site].netlify.app
   TTL: 3600 (1 hour)
   
   Option B (A Record for apex domain):
   Type: A
   Name: shop
   Value: [Netlify load balancer IP: 75.2.60.5 or 99.83.231.61]
   TTL: 3600

8. VERIFY SSL
   - Ensure HTTPS is enabled in Netlify
   - SSL certificate should auto-provision

9. TEST DEPLOYMENT
   - Visit https://shop.zedseclabs.co.za
   - Check all pages load correctly
   - Test admin panel (password: 66696)
   - Verify images display
